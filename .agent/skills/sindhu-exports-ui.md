# SINDHU EXPORTS UI GUIDELINES

## THEME: Industrial Opulence
- **Background**: #121416. Never use white backgrounds.
- **Primary (Gold)**: #f2ca50
- **Secondary (Blue)**: #3B82F6
- **Font**: Manrope only.

## STACK
Next.js 14 + Tailwind CSS + Framer Motion + Firebase Firestore + Firebase Auth.

## DESIGN SYSTEM RULES
1. **BASE**: Always dark — #121416 or darker.
2. **VIBRANT COLOR LIFE**: Glowing orbs, gradient splashes, vivid accent sections. 80% dark + 20% vivid color (Gold, Blue, Purple, Cyan ONLY).
3. **Product Images**: aspect-ratio 3/4, grayscale by default, color on hover.
4. **Pricing**: Always show both ₹ Indian rupees AND $ US dollars on every product.
5. **Headlines**: Use gradient text for major headlines.
6. **Animations**: Framer Motion scroll fadeUp on all sections. Hover scale on all cards. Glowing hover box-shadows.

## PAGES
- Landing Page
- Shopping Page (Kith.com layout + Shakawear.com product grid inspiration)
- Individual Product Page
- Cart Page
- Admin Dashboard (Firebase Auth protected)
